### 7.8.1 问题
- 来源：[LeetCode-多数元素](https://leetcode-cn.com/problems/majority-element/)
- 问题：给定一个大小为 n 的数组，找到其中的多数元素。多数元素是指在数组中出现次数大于 ⌊ n/2 ⌋ 的元素。<br>
    你可以假设数组是非空的，并且给定的数组总是存在多数元素。

### 7.8.2 示例
- 示例一
```js
输入: [3,2,3]
输出: 3
```
- 示例二
```js
输入: [2,2,1,1,1,2,2]
输出: 2
```

### 7.8.3 实现代码（js）
- 执行用时：`68ms`
- 超过：`96.67%`
```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var majorityElement = function(nums) {
    let major = nums[0];
    let count = 0;
    for (let i = 0; i < nums.length; i++) {
        if (count == 0) {
            major = nums[i];
        }
        if (major == nums[i]) {
            count ++;
        }
        else {
            count --;
        }
    }
    return major;
};
```
